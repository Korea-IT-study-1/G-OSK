# 구현한 주요기능

+ 관리자 페이지
    + 상품 등록/삭제/수정 
    + 회원 등록/삭제/수정
    + 좌석 유지보수/이동/강제퇴장
    + 상품 조회
    <br>
+ 원데이/정액권/지정석 페이지
    + 제품 종류
    + 제품 좌석선택
    + 결제
    <br>
+ 마이 페이지
    + 구매내역
    <br>
+ 입실/퇴실 페이지
    + 구매 제품에 따라 입실
    + 퇴실시 이용권종료/퇴실 선택
    <br>
+ 로그인 페이지
    + 회원가입 및 로그인
